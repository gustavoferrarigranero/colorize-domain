Política de Privacidade

Colorize Domain — Última atualização: junho de 2026

Dados coletados

Esta extensão não coleta, armazena ou transmite nenhum dado pessoal, informações de navegação ou qualquer outro tipo de informação para servidores externos.

Armazenamento local

Todos os dados criados pelo usuário (regras de domínio, seletores CSS e cores) são armazenados exclusivamente no chrome.storage.sync do navegador. Esses dados são sincronizados entre os dispositivos do usuário através da conta Google, conforme a política de privacidade do Google Chrome.

Uso de permissões

storage — utilizada exclusivamente para salvar e recuperar as regras configuradas pelo usuário.

host_permissions (*://*/*) — utilizada exclusivamente para aplicar as cores nos elementos CSS das páginas visitadas, conforme as regras definidas pelo usuário.

Compartilhamento com terceiros

Esta extensão não compartilha nenhum dado com terceiros.

Contato

Se você tiver dúvidas sobre esta política de privacidade, abra uma issue no repositório do projeto.
